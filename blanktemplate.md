# Playbook Name

## Variables required

- `my_devices`: One or more groups or host patterns, separated by colons. 
- `my_facts`: Whether to collect facts per device: `yes` or `no`.
- `my_config`: Config to apply. Ex: 

```yaml
my_config: |
  ansible code here
```
